# @airalogy/aimd-editor

`@airalogy/aimd-editor` provides AIMD authoring UI on top of Monaco and WYSIWYG tooling.

## Install

```bash
pnpm add @airalogy/aimd-editor
```

## Main Capabilities

- AIMD syntax highlighting and editor integration.
- Field insertion dialogs for `var`, `step`, `check`, `quiz`, and references.
- Configurable editing UX for protocol authoring workflows.

## Example

```ts
import { AIMD_LANGUAGE_ID } from "@airalogy/aimd-editor"

console.log(AIMD_LANGUAGE_ID)
```

For full interactive integration, refer to the `aimd/demo` editor page.
